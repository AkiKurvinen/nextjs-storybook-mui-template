#  Next.js app example with atomic design 

## Installation
```bash
npm install
```

## Run project
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to see app running.  

## Run Storybook
```bash
npm run storybook
```
Open [http://localhost:6006/](http://localhost:6006/) to see Storybook running.

GitHub Actions
